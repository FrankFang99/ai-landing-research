数据驱动 · 周更自动流水线

每周一 09:00 (UTC+8) 自动跑一次：抓公开案例 → LLM 分类 → 自动开 PR → review 后入库。本网页由同一仓库的 GitHub Actions 自动部署。

1

读 sources.yaml

信息源清单：公开网页、GitHub 仓库、RSS、公众号、视频站都可加入。

→
2

抓取新内容

web_search + GitHub API 双通道拉取过去 7 天的更新。

→
3

LLM 分类

判断是否值得入库、归到 8 行业哪个、补到哪个资产目录。

→
4

生成 PR

自动开 PR 标 weekly-update，等人工 review 后 merge。

→
5

本页重部署

merge 后 GitHub Actions 自动重新部署本页。

想加新信息源？

编辑仓库根目录的 automation/sources.yaml，加一条就行。三种类型：

- type: web
  name: 36氪 AI 频道
  url: https://36kr.com/information/AI
  keywords: [AI落地, 行业应用, 大模型]

- type: github_repo
  name: awesome-industry-ai
  url: https://github.com/xxx/awesome-industry-ai
  track: releases        # releases | commits | issues

- type: web
  name: 哔哩哔哩科技区 UP 主
  url: https://space.bilibili.com/xxx
  keywords: [AI落地, 行业案例]

提交 PR 即可，下次 cron 自动生效。
