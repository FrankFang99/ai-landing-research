## 方向 1：医学影像 AI（肺结节 / 冠脉 / 卒中）

**一句话**：用深度学习辅助影像科医生做 CT / MR / X 光 / 病理切片阅片，从「筛查」到「诊断」到「治疗规划」全流程。

**子方向**：

- **影像筛查（肺结节 / 眼底糖网 / 乳腺）**：AI 给出「可疑」提示，医生最终签字
- **影像诊断（冠脉狭窄 / 脑卒中 / 骨折）**：AI 给出量化指标（狭窄率 / 出血量 / 病灶位置）
- **手术规划（肺部 / 肝脏 / 肾脏三维重建）**：AI 自动重建 3D 模型，给外科医生术前参考

**代表案例**：

| 公司 | 场景 | 数据 | 来源 |
|---|---|---|---|
| **推想科技** | 肺结节 / 脑卒中 / 手术规划 AI | 1000+ 家医疗机构 / 全球 30+ 国家 / NMPA + FDA + CE + PMDA + UKCA 五证齐全 | 推想官网 + 中新社 + 36 氪 |
| **数坤科技** | 冠脉 CTA / 心脏 / 脑 / 胸数字医生 | 3000+ 家医疗机构 / 90% 百强医院 / 80% 目标三甲 / 13 张 NMPA 三类证（截至 2024.6）| 数坤招股书 + 中商情报网 |
| **联影智能** | CT 骨折 / 多模态影像 | 7 张 NMPA 三类证 / 母公司联影集团硬件 + AI 一体化 | 中商情报网 |
| **深睿医疗** | 肺结节 / 肺炎 AI | 13 张 NMPA 三类证（与数坤并列第一）| 东方财富网 |

**业务决策要点**：

- 业务 leader 决策时常见的判断题：「医院该不该买 AI 影像产品？怎么选？」
- 你的回答要点：先看 NMPA 三类证数量（监管合规基线）→ 再看装机医院数（市场验证）→ 最后看单证覆盖病种（产品矩阵深度）
- 借案例：**推想 5 证齐全但 8 张三类证 < 数坤 13 张三类证**，说明「证多 ≠ 市场强」，要看「证 + 装机 + 营收」三角

> **影像 AI 的真正杀手锏不是「准确率高」，而是「拿到 NMPA 三类证 + 进院渠道 + 临床主任背书」三角**。下面是从推想 / 数坤真实数据中提炼的深拆 —— 一个 AI 影像产品怎么从「技术 demo」走到「三甲日常使用」？

#### 标杆深拆 · 推想 / 数坤影像 AI「从科研到临床全周期」

> **业务 leader 30 秒版**：影像 AI 不是「图像识别准确率 95%」这么简单 —— **从科研到临床要过 5 关**：算法验证 → 多中心临床试验 → NMPA 三类证（24-36 个月 + 1500-3000 万元/证）→ 三甲主任背书 → 医院招标采购。**真正的护城河是 NMPA 证 + 主任关系 + 进院渠道，不是算法**。推想靠「5 证齐全（NMPA+FDA+CE+PMDA+UKCA）+ 1000+ 家医院」卡全球第一梯队；数坤靠「心脑胸三领域全覆盖 + 3000+ 家医院 + 80% 目标三甲」卡国内装机第一。

<div class="case-block">
<h5>🛠 它具体怎么做（Approach）—— 5 关 + 4 个落地根因</h5>
<p><strong>5 关（医疗 AI 从算法到临床的硬门槛）</strong>：</p>
<ol>
<li><strong>算法验证</strong>：准确率 95%+ 不稀奇，关键是「罕见病 / 长尾病灶」鲁棒性 —— 推想 + 数坤都在医院长期驻场采集「边缘案例」</li>
<li><strong>多中心临床试验</strong>：必须 5-10 家三甲联合验证（推想肺结节 AI 由上海长征 / 武汉同济 / 北京友谊联合）；单中心数据不能拿 NMPA 三类证</li>
<li><strong>NMPA 三类证</strong>：平均 <strong>24-36 个月 / 1500-3000 万元/证</strong>（数据来源：护城河分析 v1 + 行业公开数据），且需「多阅片者多病例」设计</li>
<li><strong>三甲主任背书</strong>：临床试验主任 + 学术发表 + 行业指南引用 —— 推想脑卒中产品由北京天坛 / 上海中山联合发表 SCI 论文</li>
<li><strong>医院招标采购</strong>：进医院要过「医院信息科 + 设备科 + 主任 + 院长」4 道关，单家三甲决策周期 6-12 个月</li>
</ol>
<p><strong>4 个落地根因（为什么 99% 的 AI 影像公司活不下来）</strong>：</p>
<ol>
<li><strong>证拿不到</strong>：三类证门槛高，没证 = 商业化为 0</li>
<li><strong>主任不信</strong>：算法强但临床弱，主任不签字 = 进不了报告</li>
<li><strong>医院不买单</strong>：单家医院预算 30-100 万元，客单价低 + 招标流程长</li>
<li><strong>报销路径没打通</strong>：AI 影像产品目前不进医保目录（按医疗服务项目收费或医院自费），商业化路径依赖医院预算</li>
</ol>

<h5>♻️ 它的可复制性（可学到什么 + 不可学什么）</h5>
<p><strong>✅ 可复制（初创公司能学的）</strong>：</p>
<ol>
<li><strong>单点突破策略</strong>：先做 1 个病种做到极致（推想肺结节 / 数坤冠脉），再横向扩展 —— <strong>不要一上来就做平台</strong></li>
<li><strong>驻场研发模式</strong>：产品和运营长期驻扎医院，摸清医生真实需求 —— 推想融资副总裁公开承认「驻场医院 3 年才摸清流程」</li>
<li><strong>多中心临床试验</strong>：5-10 家三甲联合入组，主任医师深度参与 —— 不是「买数据」而是「共建数据」</li>
<li><strong>产品矩阵策略</strong>：从「筛查 → 诊断 → 治疗规划」纵向延伸，从「胸肺 → 心脑 → 肝胆」横向扩展</li>
<li><strong>学术绑定</strong>：与主任联合发表 SCI 论文 / 联合申请专利 / 入选工信部 / 卫健委试点</li>
</ol>
<p><strong>❌ 不可复制（学不来的）</strong>：</p>
<ol>
<li><strong>历史窗口期</strong>：推想 / 数坤都成立于 2015-2017 年的 AI 医疗风口期，融资环境 + 人才储备 + 监管路径都最有利 —— 2024 年后入场难 10 倍</li>
<li><strong>三甲主任关系网</strong>：推想与上海长征 / 武汉同济合作 8-10 年，数坤与北京协和 / 阜外合作 7-8 年 —— 新进入者短期内建不起来</li>
<li><strong>全球认证体系</strong>：FDA + CE + PMDA + UKCA 四大认证每个需要 2-3 年 + 数千万元 —— 推想花 8 年才凑齐五证</li>
<li><strong>资本耐心</strong>：数坤累计融资 20+ 亿元、亏损 8 年才看到盈利曙光 —— 早期资本（远毅 / 华盖 / 五源）的耐心是关键</li>

## 方向 2：病历生成与结构化（医生口述 → EMR）

**一句话**：用 LLM 把医生口述 / 门诊对话自动转写为结构化电子病历（EMR），减少医生文书时间。

**子方向**：

- **门诊录音转病历**：医患对话 → SOAP 病历（主诉 / 查体 / 诊断 / 治疗）
- **病历质控**：检测病历缺项 / 逻辑错误 / 医疗规范
- **病历结构化**：从非结构化文本提取关键医学实体（症状 / 诊断 / 用药）

**业务决策要点**：

- 业务 leader 决策时常见的判断题：「AI 病历能减少医生多少文书时间？合规风险多大？」
- 你的回答要点：节省 30-50% 文书时间（医联 MedGPT 公开数据），但合规上 AI 不能「独立签字」—— 必须医生最终确认
- 借案例：**医联 MedGPT + 智云健康 + 讯飞晓医** 是国内病历生成三剑客，但都需「医生签字复核」+「监管备案」双保险
