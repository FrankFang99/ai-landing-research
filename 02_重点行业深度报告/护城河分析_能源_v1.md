# 护城河分析 · 能源行业 v1

> **行业子方向**：石油 / 电力 / 新能源 / 储能（4 子方向）
> **版本**：v1（2026-06-11）
> **作者**：AI 咨询分析师
> **目标读者**：咨询公司老板 / team owner
> **时效性**：截至 2026 年 Q2
> **核心判断**：能源是"重资产 + 强监管 + 难转型但必须转"的典型——旧护城河最深（资源 / 牌照 / 央企关系），AI 拆解速度比医疗 / 法律慢 3-5 年，但**接单金线已出现**：央企信创 + 碳数据 + 虚拟电厂调度

---

## 一、旧护城河解剖（5 种）

能源行业的护城河是全行业最深的——这是 60 年央企 + 30 年改革开放沉淀下来的结构性壁垒，AI 拆不动根基，但能拆"经验层 + 服务层 + 流程层"。

### 1. 资源垄断（最硬 / 拆不动 / 接单绕着走）

**形态**：石油（中石油 / 中石化 / 中海油 / 延长石油）拥有 70%+ 国内陆上 + 海上油气探矿权 + 采矿权；煤炭（神华 / 中煤 / 兖矿）握有 80%+ 优质煤矿；电力（国家电网 / 南方电网）垄断输配电网；盐湖锂（西藏矿业 / 盐湖股份）握有青海察尔汗 / 西藏扎布耶等核心盐湖采矿权。

**护城河强度**：⭐⭐⭐⭐⭐（5/5）
- 探矿权 / 采矿权是国家发改委 + 自然资源部联合审批，单矿权审批周期 5-10 年，单矿权价值百亿级
- 油气区块"招拍挂"自 2017 年才部分放开，**实质仍是央企 + 地方国企游戏**
- 锂矿 / 稀土 / 铀矿 → 战略性矿产，外资 + 民营准入门槛极高

**AI 拆解能力**：❌ 拆不动
- 资源垄断是物理 + 制度双重护城河，AI 既不能挖矿也不能发采矿证
- 唯一影响：AI 可让"找矿"提速（地震数据解释 / 遥感找矿），但矿权仍在央企手里

**接单判断**：❌ 资源端不是接单点。咨询公司能做的只有"找矿提速 AI 工具的私有化部署"——客户是央企勘探部门，客单价 500-3000 万。

### 2. 牌照 + 央企准入（最稳 / 政策驱动 / 周期性强）

**形态**：
- **电力牌照**：发电牌照（5 大 + 4 小 + 地方国企 + 民营新能源）、输配电牌照（国网 / 南网独家）、售电牌照（2015 年改革后 3000+ 家）
- **油气牌照**：原油进口资质（21 家）、成品油出口资质（12 家）、成品油批发 / 零售（民营已放开但新建加油站审批极严）
- **新能源牌照**：新能源汽车生产资质（2017 年后基本停发，已有资质是稀缺资源）、充电桩运营牌照（部分城市要求）、储能并网调度权（电网说了算）
- **碳牌照**：CCER 重启后第三方审定 / 核证机构（仅 6 家）、全国碳排放权交易资质（2024 年扩容到钢铁 / 水泥 / 铝）

**护城河强度**：⭐⭐⭐⭐⭐（5/5）
- 牌照的本质是"政府准入"，与"是否垄断"无关——**AI 永远拿不到牌照**
- 牌照有效期 5-20 年，续期看合规
- 民营即使拿到牌照也常被"歧视性调度"（电网优先调度自有电厂）

**AI 拆解能力**：❌ 拆不动
- AI 能做的是"帮助企业更高效满足牌照要求"——如 ESG 报告自动化、双碳核算自动化、监管报送自动化
- 创造**"牌照合规 AI 助手"** 商业模式——但客户是已持牌企业，客单价 200-1000 万

**接单判断**：✅ **央企信创 + 牌照合规** 是接单金线之一。

### 3. 重资产（最深 / 折旧刚性 / AI 提效不颠覆）

**形态**：
- **油田**：单油田投资 100-1000 亿（长庆 / 塔里木 / 渤海），钻井 + 集输 + 处理厂
- **电网**：国家电网 2024 年资产 5.2 万亿，南方电网 1.3 万亿，单特高压工程 200-500 亿
- **电站**：单核电机组 200 亿、单百万千瓦煤电 50 亿、单特大型水电站 1000 亿+
- **新能源**：单陆上风电场 30-100 亿、单海上风电场 100-300 亿、单 GW 级光伏电站 30 亿
- **储能**：单 100MW/200MWh 共享储能电站 5-10 亿

**护城河强度**：⭐⭐⭐⭐⭐（5/5）
- 20 年折旧周期 → 资产退不出去
- 沉没成本 → 央企"保产能"动机极强
- 资本壁垒 → 民营 + 外资根本进不来

**AI 拆解能力**：⚠️ 部分拆解
- AI 能做"资产数字化 + 设备预测性维护 + 智能调度"——这是国家电网"能源大脑"、中石油"勘探开发梦想云"在做
- 提效幅度 5-15%——能省成本但**改变不了资产的所有权和折旧结构**
- 单电站接入 AI 调度系统年节省 100-500 万 → 看似 ROI 不错但**绝对值相对 100 亿资产太小**

**接单判断**：✅ 设备预测性维护 + 智能巡检是高 ROI 接单点。客户是发电集团（华能 / 大唐 / 国电 / 华电 / 国电投）三级公司。

### 4. 渠道（最深 / 民营破不了 / AI 也破不了）

**形态**：
- **加油站**：中石化 3 万座 + 中石油 2.2 万座（占全国 50%），覆盖高速 / 国道 / 县乡；民营加油站 9 万座但单店油源受两大集团控制
- **电力营业厅**：国网 2.4 万个营业厅 + 南网 8000 个，乡镇覆盖 100%
- **加气站 / 充电站**：中石油 / 中石化 2022 年起大举建充电站，借用加油站渠道；国家电网"光储充"换电站布局
- **工业园电网接入**：高压接入必须经电网审批，5 大发电集团参股工业园区分布式能源项目

**护城河强度**：⭐⭐⭐⭐（4/5）
- 加油站 / 营业厅是**几十年物理网络**，新建一个加油站的审批 + 拿地周期 3-5 年
- 渠道绑定用户（车牌 = 加油习惯；电表 = 用电习惯）
- AI 破不了"网点密度"——但 AI 能让**渠道运营效率**提升 10-20%

**AI 拆解能力**：⚠️ 部分拆解
- AI 可做"加油站非油业务推荐"（咖啡 / 便利店 / 车后服务），提升单店 GMV 10-30%
- AI 可做"营业厅智能客服 + 业务办理自动化"，降低人工成本
- AI 可做"充电桩智能选址 + 智能调度"

**接单判断**：✅ 加油站 / 充电站 / 营业厅"AI 提效"是高 ROI 接单点。客单价 100-1000 万。

### 5. 央企关系（最软性 / 但最核心 / 咨询公司唯一能做的）

**形态**：
- **决策链**：能源央企（特别是 5 大发电 + 3 大石油 + 2 大电网）的 IT / 数字化采购要走"集团总部 → 二级公司 → 三级公司"三级审批，最终由集团科技部 / 数字化部拍板
- **采购周期**：单一项目从立项到签约 12-18 个月，验收后还要 1-2 年质保
- **决策人**：集团一把手（董事长 / 总经理） + 分管副总经理 + 科技部 / 数字化部部长 + 业务部门负责人（发电部 / 营销部 / 财务部）
- **采购偏好**：倾向国资背景供应商（央企子公司 / 国资控股上市公司），倾向"长期合作 + 战略级"而非"低价中标"
- **合规要求**：信创（国产化芯片 + 操作系统 + 数据库 + 中间件）已成为硬性要求，国资委 79 号文 + 金融 / 能源 / 电信行业"2+8"信创试点

**护城河强度**：⭐⭐⭐⭐（4/5）
- 民营企业几乎拿不到头部央企的核心数字化项目
- 外企（微软 / 谷歌 / 亚马逊 / IBM）已基本退出中国能源市场
- 民营科技公司能拿到的多是"二级公司试点项目"，客单价 100-500 万

**AI 拆解能力**：❌ 完全拆不动
- 这是**社会资本 + 政治资本 + 关系资本**的复合护城河，AI 替代不了"请客吃饭"和"长期信任"
- 但 AI 可做"央企决策辅助"——**把政策文件 / 领导讲话 / 行业研报 AI 化，输出"建议书 / 立项书 / 可研报告"**——这是咨询公司的金线

**接单判断**：✅✅✅ **咨询公司最高 ROI 接单点**——"央企决策辅助 AI"+"信创合规 AI 助手"+"立项可研 AI 生成"。客单价 200-2000 万。

## 二、AI 拆解路径（4 维度）

### 1. 替代（AI 替代人 / 经验层被折叠）

**电网调度 AI 替代经验调度员**：
- 国家电网"能源大脑"（"AI 调度员"）已在华东 / 华中试点，2024 年覆盖国网 27 家省级公司
- 替代逻辑：传统调度员靠 10-20 年经验应对电网波动，AI 通过强化学习 + 历史数据训练后，处理 95% 标准化调度场景（**剩余 5% 极端工况仍需人**）
- 节省人力：单省级电网调度中心 50-100 名调度员，AI 替代后减员 30-50%
- 数据：**国家电网"能源大脑"项目**自 2020 年立项，2024 年覆盖 27 家省级公司，调度自动化率达 95%（来源：国家电网 2024 年数字化白皮书）

**油田地震数据 AI 解释**：
- 中石油"勘探开发梦想云"AI 地震解释系统已在中国 15 个主力油田应用
- 替代逻辑：传统地震解释靠物探工程师人工标注，单区块解释周期 3-6 个月；AI 解释将周期压缩到 1-2 周
- 数据：**中石油"勘探开发梦想云"**累计解释地震数据 200+ PB，覆盖 15 个主力油田（来源：中石油 2024 年数字化转型报告）

**节省成本**：单油田节约物探工程师 30-50 人 × 年薪 50 万 = 1500-2500 万/年

### 2. 摊薄（AI 让边际成本下降 / 重资产回报率提升）

**新能源车充电 AI 调度**：
- 蔚来"Power Up Plan"、小鹏"超充网络"、特斯拉"V4 超充"都在用 AI 优化充电桩功率分配、电网负荷平衡、峰谷电价套利
- 摊薄逻辑：充电桩固定资产投入 50-100 万/根，单桩利用率从 30% 提到 50% → ROI 翻倍
- 蔚来 2024 年公开数据：AI 调度让换电站单站日均服务量提升 20%

**储能电池 AI 优化**：
- 阳光电源"PowerTitan"储能系统内置 AI 优化，AI 动态调整充放电策略延长电池寿命
- 摊薄逻辑：储能电池循环次数从 6000 次提升到 8000 次 → 电池更换周期延长 3 年 → 单电站生命周期成本下降 15-20%
- **阳光电源 2024 年报披露**：AI 优化让储能项目 IRR 提升 2-3 个百分点

### 3. 绕过（AI + 分布式让央企护城河失效）

**分布式光伏 + 储能绕过集中式电网**：
- 工商业屋顶光伏（隆基 / 晶澳 / 天合）+ 用户侧储能（宁德时代 / 比亚迪 / 阳光）+ 智能微电网 → **让工厂"自发自用、余电上网"**
- 绕过逻辑：传统电网靠"发电 → 输电 → 配电 → 售电"四级赚取差价，分布式光伏让"自发自用"部分完全不经过电网
- 隆基绿能 AI 硅片切割：单晶硅片切割损耗从 0.3mm 降到 0.2mm → 硅片出片率提升 5% → **单 GW 组件硅料成本下降 500-1000 万**

**虚拟电厂绕过传统电厂**：
- 国家电投"天枢一号"虚拟电厂平台已聚合 2000+ 分布式资源（分布式光伏 / 储能 / 可调负荷），相当于 1 个百万千瓦电厂
- 绕过逻辑：传统电厂是"集中式固定资产"——建 1 个百万千瓦电厂要 50 亿 + 5 年；虚拟电厂是"聚合分布式资源"——零固定资产 + 1-2 年
- **国家电投"天枢一号"** 2024 年公开：聚合容量 5GW，可调容量 2GW，年调度电量 30 亿 kWh（来源：国家电投 2024 年 ESG 报告）

### 4. 加速（AI 让监管 / 合规流程提速）

**碳排放核算**：
- 全国碳市场 2024 年扩容到钢铁 / 水泥 / 铝，覆盖 5000+ 家企业
- AI 让"碳排放核算"从人工月报 → AI 实时核算 + 自动报送
- **节省人力**：单企业碳排放管理员 1-2 人 → AI 替代后 0.5 人 → 年节省 50-100 万
- 市场容量：5000 家企业 × 客单价 100-500 万 = 50-250 亿

**ESG 报告**：
- A 股 500 强 + 港股 + 海外上市公司 3000+ 家强制披露 ESG
- AI 让"ESG 报告生成"从人工季报 / 年报 → AI 实时生成 + 第三方核证
- **节省人力**：单上市公司 ESG 团队 3-5 人 → AI 替代后 1-2 人 → 年节省 200-500 万
- 市场容量：3000 家 × 客单价 200-1000 万 = 60-300 亿

## 三、新护城河（5 种）

能源行业的新护城河比医疗 / 物流更深——**因为能源的数据是物理世界的真值**（不像医疗是医院各自割据），央企一旦建立数据飞轮，复制难度极高。

### 1. 数据飞轮（电网 + 充电 + 用户 三位一体）

**形成路径**：
- 国家电网掌握 5.7 亿用户的实时用电数据 + 11 亿智能电表 + 27 家省级电网的 SCADA 数据
- 国家电网"能源大脑"接入这些数据后，AI 模型精度持续提升 → 调度准确率从 95% 提升到 98%
- **新进入者拿不到这些数据**——电网数据是国安级保密，民营 / 外资 AI 公司无门

**护城河强度**：⭐⭐⭐⭐⭐（5/5）
- 数据是物理世界真值，不像电商数据是用户行为（可爬取）
- 央企数据 + 国资委管控 → 民营公司 5-10 年内拿不到
- **是央企在 AI 时代最深的护城河**

**接单金线**：✅ 帮国网 / 南网建设"能源大脑 2.0"（融合充电 + 储能 + 虚拟电厂数据）

### 2. 算力调度（西部风光电 + 东部算力中心）

**形成路径**：
- 中国"东数西算"工程：8 大算力枢纽中 4 个在能源富集区（宁夏中卫 / 内蒙古和林格尔 / 甘肃庆阳 / 贵州贵安）
- 西部风光电便宜（0.15-0.25 元/kWh），东部算力中心贵（0.5-0.8 元/kWh）
- **AI 算力调度**：用 AI 把东部算力任务调度到西部数据中心 → 单算力任务成本下降 50%
- 阿里云"张北算力中心"、腾讯"贵州数据中心"、华为"贵安数据中心"都已落地

**护城河强度**：⭐⭐⭐⭐（4/5）
- 算力调度需要"绿电 + 算力 + 网络"三位一体，仅头部云厂能做
- 中小云厂 + AI 公司拿不到便宜的绿电（直购电资格 + 输配电价补贴）

**接单金线**：✅ 帮云厂（阿里 / 腾讯 / 华为 / 字节）建设"绿电直购 + 算力调度"AI 系统。客单价 1000-5000 万。

### 3. 储能 + AI 调度（虚拟电厂）

**形成路径**：
- 中国新型储能装机 2024 年底达 73.76 GW（来源：国家能源局 2025 年 1 月新闻发布会），年增 50%+
- 储能 ≠ 简单电池堆——储能需要"AI 调度"决定何时充电 / 放电 / 待机
- 阳光电源 / 宁德时代 / 比亚迪 / 华为数字能源都在做"储能 AI 调度"
- **虚拟电厂**：把分布式储能 / 光伏 / 可调负荷聚合为 1 个"电厂"参与电网调度
- 国家电投"天枢一号"是行业标杆

**护城河强度**：⭐⭐⭐⭐（4/5）
- 储能 AI 调度需要"储能硬件 + AI 算法 + 电网调度协议"三位一体
- 阳光电源 / 宁德时代等头部已建立硬件壁垒，AI 算法 + 调度 know-how 是软件壁垒
- **新进入者只能做"虚拟电厂聚合商"**（如国家电投 / 冀北电力 / 山西晋能）——客单价 500-3000 万

**接单金线**：✅ 帮新能源集团（华能 / 大唐 / 国电 / 华电 / 国电投 / 三峡 / 中广核）建设虚拟电厂 AI 调度平台

### 4. 碳数据 + ESG know-how

**形成路径**：
- 全国碳市场扩容 + 欧盟 CBAM（碳边境调节机制）2026 年正式征收 + ESG 披露强制化
- 碳数据 = "碳排放核算 + 碳资产管理 + CCER 开发 + 碳金融 + ESG 披露"全链条 know-how

**护城河强度**：⭐⭐⭐⭐（4/5）

- **AI 时代后，know-how 不再是人的经验，而是"AI 模型 + 训练数据 + 反馈闭环"**

**接单金线**：✅ 帮上市公司（3000 家）建设"碳数据 + ESG"AI 系统。客单价 200-1000 万。

### 5. 央企数字化项目（信创 + 国产化）
