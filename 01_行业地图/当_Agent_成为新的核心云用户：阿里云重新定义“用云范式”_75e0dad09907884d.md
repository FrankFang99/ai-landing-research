# 当 Agent 成为新的核心云用户：阿里云重新定义“用云范式”

## 摘要

阿里云提出当 AI Agent 成为核心云用户后，云服务模式将从「人找服务」转变为「Agent 自主调度」，云厂商需重新定义 API 定价、计量方式和安全架构，包括按 Token 计费向按任务计费演进、模型即服务向工具链服务延伸等趋势。
