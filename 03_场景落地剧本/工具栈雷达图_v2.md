## 一、评分维度说明

### 上手难度（★-★★★）
- **★**：1 周内可上手（API 调用 + 基础微调）
- **★★**：2-4 周可上手（需了解架构 + 中等微调）
- **★★★**：1-3 个月可上手（需深度定制 + 复杂集成）

### 成本（人民币/月，按典型使用量）
- **免费 / 开源**：¥0
- **低**：¥1-10K
- **中**：¥10K-100K
- **高**：¥100K+
- 注：成本按"中等规模企业（100-500 人 / 日均 10K-100K 调用）"估算

### 推荐度（★-★★★）
- **★**：可用但不推荐（合规 / 性能 / 成本有硬伤）
- **★★**：主流选择（70% 场景可考虑）
- **★★★**：强烈推荐（特定场景最优解）

## 十、工具栈选型决策树（销售 / 驻场必备）

```
客户是哪个行业？
├─ 银行/保险
│   ├─ 是否央国企？→ 智谱 GLM-4.6 + 华为盘古

│   ├─ 是否央国企？→ 华为盘古制造大模型 + 昇腾 Atlas 500
│   ├─ 是否腰部企业？→ YOLO11 + 阿里通义工厂大模型
│   └─ 是否做具身智能？→ 优必选 Walker S1（观望）
├─ 物流
│   ├─ 是否头部（顺丰/京东/菜鸟）？→ 顺丰丰知大模型 / 京东物流 / 菜鸟自研
│   ├─ 是否中型区域分拣中心？→ DeepSeek-V3 + CrewAI 0.80 + 极智嘉
│   └─ 是否做无人配送？→ 京东无人车 / 九识智能
├─ 政务
│   ├─ 是否省级政府？→ 中国移动"九天·海算" + 阿里通义
│   ├─ 是否市级政府？→ DeepSeek-V3（辽宁模式） + 飞书/钉钉
│   └─ 是否委办局？→ 阿里百炼（轻量级） + 飞书
└─ 能源电网
    ├─ 是否国网/南网？→ 光明电力大模型 / 大瓦特
    └─ 是否省网/地市？→ 华为盘古电力大模型 + 昇腾 Atlas 500 + 鸿蒙系统
```
