## 六、使用指南（按角色）

###老板（战略决策）

1. **季度回顾**：每季度看 `04_甲方决策评估/老板战略路线图_v2.md` 的6象限 +18 个月季度路线图，识别行业优先级变化
2. **接单判断**：每个新客户跑8行业重投优先级总表 +6象限决策矩阵 +3 接单金线匹配
3. **资源配置**：55% 重投 v15行业 +25%试点新增3行业 +12%央企信创 +5%监管资质 +3%储备

###销售（打单一线）

1. **打单前 30 分钟**：跑客户所在行业的剧本 v2（5 行业）+ 护城河深度（v3 / v1）+ BattleCard v2（8 行业）+ 真实案例
2. **打单金线匹配**：根据客户画像匹配3 条金线之一（核心 /监管 /央企信创）
3. **拒绝话术**：从伪需求识别 v2 的16 条预警 +12 条拒绝客户画像选3-5 条对应

###驻场（交付一线）

1. **入场前 2 小时**：读客户行业护城河深度（v3 / v1）+ 剧本 v2（5 行业）+ BattleCard v2（8 行业）+ 5A 落地清单 + 烂尾项目警示录
2. **入场1 周内**：沉淀行业 know-how 进 `06_案例库/案例库使用指南_v2.md`
3. **退出前**：用伪需求 v2 看客户是不是伪需求，决定是否承接续费

###行业专家 /咨询师

1. **新增行业研究**：直接套用4步法 +5评分模板（详 `01_行业地图/第三轮研究方法论_v1.md`）
2. **数据校准**：每6 个月重新评估8行业评分 +6象限位置
3. **方法论 v2**：建议补量化打分模型 / 海外案例 / AI Agent 经济 /监管口径量化
