# 老板战略路线图 v2（18 个月可执行版 · 8 行业 [5+3] 重投优先级）

> **作者**：各行业企业主代表
> **版本**：v2.0（2026-06-11 — 第四轮收尾）
> **升级定位**：v1 是 5 行业 12 个月版。v2 在 v1 基础上**行业 5 → 8（保留 v1 锁定的法律/财税/餐饮/宠物/物流仓储，新增医疗/能源/教育 3 行业）/ 决策矩阵 4 象限 → 6 象限（新增监管准入 + 政策窗口期 2 维度）/ 接单金线 1 条 → 3 条（保留核心金线，新增监管合规 know-how + 央企信创）/ 季度路线图 12 个月 → 18 个月（因医疗 NMPA 准入 / 能源央企准入 / 高校招标周期更长）**。
> **核心问题**：**我作为老板，从 2026-07 到 2027-12 这 18 个月，把 1600 万总盘（1500 万行业累加 + 100 万兜底储备）、6 个 agent 团队，按什么节奏打 8 行业、接哪 3 类金线、避哪些雷、怎么兜底？**
> **横向证据基线**：8 份核心交付物交叉验证（行业地图 v2 / 案例库 v2 / Top5 护城河 v2 / 剧本 v2 / 红队 v2 / 方法论 v1 / 路线图 v1 / 新增医疗能源教育 3 份护城河 v1）。

## 0. 老板的"一句话战略"（先看这页）

> **18 个月只做 5 件事**：①**重投 v1 锁定的 5 个咨询友好行业**（法律 / 财税 / 餐饮 / 宠物 / 物流仓储 = 55% 资源）；②**接单金线 1 → 3 条**（核心金线 + 监管合规 know-how + 央企信创）；③**试点新增 3 个"慢热但护城河最深"行业**（医疗 / 能源 / 教育 = 25% 资源，18 个月准备期）；④**用 6 季度建 4 类新护城河**（行业 know-how + 监管合规 + 央企信创 + 数据飞轮）；⑤**守住 3 红线 + 12 拒绝画像 + 3 行业专属红线**。
>
> **资源分配（总盘 1600 万 / 18 个月 = 1500 万行业累加 + 100 万兜底储备）**：55% 重投 v1 五行业 + 25% 试点新增三行业 + 12% 央企信创 + 5% 监管合规资质 + 6.25% 兜底储备。
>
> **3 个行业专属红线**：① 医疗不接"AI 独立问诊 / AI 开方"（NMPA 红线）；② 能源不接"民营油企 + 顶层战略"（央企关系壁垒）；③ 教育不接"纯 K12 学科类辅导"（双减红线）。
>
> **3 条接单金线**：① **核心金线**——行业 know-how + 客户深度陪跑；② **监管合规 know-how**——医疗 NMPA / 教育双减 / 政务采购 / 金融银保监的合规能力；③ **央企信创**——能源/医疗/教育央国企的国产化 AI 私有化部署。
>
> **复盘节奏**：每季度回 1 次（季度末看落地 vs 预期），半年大调 1 次（接单命中率 < 30% 触发收缩），年底全调 1 次（结合 3 年压测重新排序）。
