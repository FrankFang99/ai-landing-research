## 详细描述

观远数据(GuanData)是中国 BI / 决策智能赛道的代表性厂商,创始人 & CEO 苏春园在 2018 年首次公开提出"5A 落地路径"方法论,经过 7 年 7 届年度智能决策峰会的迭代,已经成为中国数据 BI 行业的"共识方法论"。

**5A 原始 5 个词(按观远官方顺序)**:
> **Agile(敏捷化)** → 从经营核心场景切入,快速构建数据指标体系
> **Accurate(场景化)** → 针对"人货场"关键场景的多维指标体系
> **Automated(自动化)** → 用户端自助分析 + 业务端"数据追人"
> **Actionable(行动化)** → 从数据到行动,决策闭环
> **Augmented(增强化)** → AI + 深度学习 + 预测 / 诊断

**5A 演进到 2024-2025 的 Agentic BI 框架**:苏春园在 2025 智能决策峰会上,首次把 5A 与 Agentic AI 融合,提出"AI+BI 渐进式落地"路径:
- 1.0 阶段(2018-2020):自助 BI + 指标体系
- 2.0 阶段(2021-2023):ChatBI + 自然语言问数
- 3.0 阶段(2024-2025):Agentic BI + AI 主动追人

**2025-10-22 上海"数智·渐进力"峰会核心成果**:苏春园发布 GA 4 款核心产品:
1. **BI Anywhere**(移动端 BI,2025 升级版)
2. **BI 决策**(决策智能平台)
3. **Chat BI**(自然语言问数)
4. **BI Agentic**(AI 主动追人 + 智能体协同)

**行业启示(销售打单核心要点)**:
- **小步快跑**:从最小业务场景(单店 / 单品 / 单指标)切入,3 个月出可衡量结果
- **优先高频可量化场景**:选"每天发生 + 能用数据衡量 + 业务愿意为结果买单"的场景
- **业务主导 + IT 赋能**:业务部门提需求,IT / 数据部门建底座,避免"技术视角主导"
- **拒绝"集团级大模型"陷阱**:5A 强调"从最小场景切入",正好是失败案例 #2"3000 万烧光 CTO 被裁"的对立面

## 关键洞察

- **能教我们的**:
  1. **5A 是"中国 BI 行业共识方法论"**——苏春园 2018 首次公开后,5A 已经被瓴羊 / 帆软 / 网易数帆 / 永洪等多家 BI 厂商参考,具有"行业锚点"地位
  2. **7 届峰会主题的迭代就是中国 BI 行业的演进史**——从"决策大脑"(2018)→"敏捷向未来"(2023)→"数智渐进力"(2025),反映了从"理论框架"到"渐进式落地"的演化
  3. **"渐进式落地"是反"集团级大模型"陷阱的最佳武器**——观远在客户落地中坚持"3-6-12 月分阶段建设"路径(3 月单场景 POC,6 月扩展到 3-5 场景,12 月规模化)
  4. **Agentic BI 是 2025-2026 的关键演进方向**——苏春园认为"BI 的下一个十年是 Agentic BI",从"人问数"到"数追人"是范式跃迁
  5. **4 大客户案例(连锁零售 / 歌力思 / 西瓜创客 / 联合利华)覆盖了不同行业 + 不同规模 + 不同阶段**——销售可以根据客户画像精准选案例

- **可能失败的环节**:
  1. **5A 是"概念框架"不是"实施清单"**——具体到客户场景,5A 怎么落地需要顾问团队深耕
  2. **观远自有产品 BI Anywhere / Chat BI 等与"5A 方法论"绑定较强**——其他 BI 厂商的客户不一定适用观远产品
  3. **客户问"5A 跟市面上 N 种方法论有什么区别"时,5A 的差异化不够明显**——苏春园本人在 2023 峰会也承认"5A 一直在演进"
  4. **"5A 出处"在 Datafun 社区未找到,可能在 CIO 时代学院大会 + 亿邦动力 + 观远自有峰会**—— task 里"Datafun"可能是团队记忆偏移,实际出处见上述 URL
  5. **"联合利华 2018 → 2024 悟空系统"的长程合作数据来自观远自披露**——客户方(联合利华)未独立公开核实,销售引用时建议交叉验证
